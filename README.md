


# 🚀 30 days css challenge

## 🧐  Sobre o desafio

Serão 30 dias fazendo mini projetos diários com HTML5 e CSS3 visando melhorar a habilidade de codificação. Este desafio foi inspirado pela [@Mitsu325](https://github.com/Mitsu325/CSS_Challenge#sobre!)

## 📜 Regras 

-   Realizar um projeto por dia
-   O projeto deve ser concluído até 23:59
